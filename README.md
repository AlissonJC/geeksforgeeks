My solutions to [**GeeksforGeeks**](https://www.geeksforgeeks.org/) practice
problems, written in modern C++.

Each solution lives in [`solutions/`](solutions/) as `solutions/<problem-slug>.<ext>`,
matching the slug in the problem URL — e.g. `solutions/reverse-a-string.cpp` for
`https://www.geeksforgeeks.org/problems/reverse-a-string/1`. Every file includes a
header explaining the problem, the chosen approach, and the time/space complexity.
